DEPLOYMENT OPTIONS
==================

DEVELOPMENT:
  python app.py

PRODUCTION - GUNICORN:
  gunicorn --worker-class eventlet -w 1 \
    --bind 0.0.0.0:5000 app:app

PRODUCTION - DOCKER:
  FROM python:3.11
  COPY requirements.txt .
  RUN pip install -r requirements.txt
  COPY . .
  CMD ["gunicorn", "--worker-class", "eventlet", \
       "-w", "1", "--bind", "0.0.0.0:5000", "app:app"]

KUBERNETES:
  Uses /api/deployment/readiness and 
  /api/deployment/liveness probes for health checks

TESTING:
  pytest tests/ -v --cov
