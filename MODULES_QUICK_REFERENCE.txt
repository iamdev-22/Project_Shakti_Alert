QUICK START
===========

To use all modules:

1. Import in app.py:
   from shakti_alert.config import get_config
   from shakti_alert.logging_setup import setup_logging
   from shakti_alert.validators import AlertRequest
   from shakti_alert.error_handler import handle_exceptions
   from shakti_alert.enhancements import ResponseBuilder
   from shakti_alert.db_migration import DatabaseMigrator
   from shakti_alert.performance_monitor import PerformanceTracker
   from shakti_alert.deployment_manager import deployment_bp

2. Initialize:
   config = get_config('production')
   logger = setup_logging(config.LOG_LEVEL)
   db_migrator = DatabaseMigrator(config.DATABASE_PATH)
   tracker = PerformanceTracker()

3. Register blueprints:
   app.register_blueprint(deployment_bp)

4. Use in routes:
   @app.route('/api/test', methods=['POST'])
   @handle_exceptions
   @track_request('/api/test')
   def test_route():
       data = AlertRequest(**request.json())
       return ResponseBuilder().success({"status": "ok"})

All modules are fully functional and can be integrated immediately.
