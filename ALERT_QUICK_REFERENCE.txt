MOST COMMON ISSUES
==================

Problem 1: "NO GUARDIANS FOUND!"
→ No guardians configured
→ Fix: Edit config.json and add guardians
→ Format: {"name": "Mom", "phone": "+919876543210"}

Problem 2: "ConnectionError"
→ WhatsApp server not running
→ Fix: Run "npm start" in wa_server folder
→ Check port 3001 is free

Problem 3: "FAILED - HTTP 401"
→ WhatsApp authentication issue
→ Fix: Check server logs for auth errors
→ May need to re-authenticate WhatsApp

Problem 4: Alert doesn't send twice in a row
→ This is by design (prevents duplicates)
→ Wait 20+ seconds before next alert

KEY IMPROVEMENTS
===============

✅ GUARD 1: Prevent same help command from sending twice
   - Each help command gets unique ID
   - Can't be triggered multiple times
   - Only sends once per user request

✅ GUARD 2: Global cooldown (20 seconds minimum)
   - Wait at least 20 seconds between alerts
   - System-wide protection
   - Prevents alert spam

✅ GUARD 3: Token-based deduplication
   - Each alert has unique token
   - Can't be processed twice
   - Even if request is repeated

✅ Enhanced logging
   - Every step is printed
   - Easy to follow in terminal
   - Clear success/failure messages

✅ Better error handling
   - Catches all exceptions
   - Shows error types
   - Displays error messages
