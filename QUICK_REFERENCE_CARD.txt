QUICK TROUBLESHOOTING
======================

Problem: Alert not sending
Fix 1: Is WhatsApp server running? → npm start
Fix 2: Are guardians added to config.json? → Add them
Fix 3: Run: python DEBUG_ALERT_SYSTEM.py

Problem: Multiple alerts still not working
Fix 1: Restart Flask backend
Fix 2: Check terminal shows "cooldown removed"
Fix 3: Make sure each HELP is different (not duplicate)

Problem: Email not sending (but configured)
Fix 1: Check .env has GMAIL_USER
Fix 2: Check .env has GMAIL_APP_PASSWORD
Fix 3: Restart Flask after adding credentials
Fix 4: Check Gmail app password valid (refresh if needed)

Problem: Ghost emails sending (not configured but sending)
Fix: This won't happen now ✅
     Email only sends if explicitly configured
