SETUP WITH EMAIL (Optional)
===========================

Step 1: Open .env file in project root
   └─ If doesn't exist, create it

Step 2: Add these lines:
   GMAIL_USER=your-email@gmail.com
   GMAIL_APP_PASSWORD=your-app-password

Step 3: Restart Flask app
   └─ Stop and start the backend

Step 4: Test
   ├─ Trigger alert
   ├─ Check terminal: should show "[EMAIL] ✅ Sent to..."
   └─ Check your email inbox


GET APP PASSWORD (Gmail)
=========================

1. Go to: https://myaccount.google.com/security
2. Enable 2FA if not enabled
3. Find "App passwords" option
4. Select: Mail and Windows
5. Google generates 16-character password
6. Copy it to GMAIL_APP_PASSWORD in .env
