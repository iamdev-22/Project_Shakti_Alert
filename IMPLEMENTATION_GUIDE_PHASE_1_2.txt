Test 1: Email Only Sends When Configured
─────────────────────────────────────────

Scenario A: No GMAIL credentials
   1. Ensure GMAIL_USER not in .env
   2. Ensure GMAIL_APP_PASSWORD not in .env
   3. Send alert
   Expected: Email NOT sent ✅
   Terminal: Shows "[EMAIL] ❌ Gmail config missing"

Scenario B: GMAIL configured but alert emails disabled
   1. Set GMAIL_USER in .env
   2. Set GMAIL_APP_PASSWORD in .env
   3. Call: PUT /api/email-preferences/alert with {"enabled": false}
   4. Send alert
   Expected: Email NOT sent ✅
   Terminal: Shows "[📧] Alert emails disabled by user"

Scenario C: GMAIL configured and alert emails enabled
   1. Set GMAIL_USER in .env
   2. Set GMAIL_APP_PASSWORD in .env
   3. Call: PUT /api/email-preferences/alert with {"enabled": true}
   4. Send alert
   Expected: Email sent ✅
   Terminal: Shows "[📧] Email sent successfully"
